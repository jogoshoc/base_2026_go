Informações de publicação na Web

Os comentários HTML nesta página contém as informações de configuração que permitem aos usuários editar páginas na sua Web usando o 'Assistente para publicação na Web da Microsoft' ou programas que usem este assistente, como o FrontPad, usando o mesmo nome de usuário e senha que eles usariam se estivessem criando no Microsoft FrontPage. Consulte o Internet SDK da Microsoft para obter mais informações sobre as APIs do 'Assistente para publicação na Web'.
